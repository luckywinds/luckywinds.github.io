import语法

ImportDeclaration = "import" ImportSpec
ImportSpec        = [ "." | "_" | Identifier ] ImportPath
  • .: 匿名别名，可以直接使用包的方法，无需指定包名
  • _: 仅使用包的init函数，其他方法不使用
  • Identifier: 显式别名，代替原始包名字

示例：

import (
    "math"
    m "math"
    . "math"
    _ "math"
    "go.etcd.io/etcd/clientv3"
)
  • 短路径：标准库，例如：“math”
  • 长路径：第三方库，例如：“go.etcd.io/etcd/clientv3”
