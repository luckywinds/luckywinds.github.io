Gateway边缘控制

Ingress

入口流量控制

  1. 外部流量通过LB进入集群
  2. 集群通过节点NodePort开放入口
  3. 流量通过Istio Gateway进入服务网格内
  4. Gateway通过Envoy代理实现
  5. Envoy通过Virtual Service规则实现流量路由
  6. Virtual Service根据规则匹配对应的Service

Egress

出口流量控制

Egress类似Ingress，通过Envoy提供外部访问代理
